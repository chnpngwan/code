在上面的示例中，使用了 Bootstrap 框架的 Carousel（轮播）组件来创建轮播图。 首先，在标签中引入 Bootstrap CSS 文件，然后使用
元素创建一个 Carousel 容器， 并设置id为"myCarousel"。轮播指示器使用
    元素创建，并在每个指示器上设置data-target属性为 Carousel 容器的id， data-slide-to指定了每个指示器对应的幻灯片索引。 轮播内容使用
    元素和.carousel-item类创建，其中第一个幻灯片添加了.active类， 表示初始时显示该幻灯片。每个幻灯片中使用标签加载图片。 前进/后退按钮使用元素和.carousel-control-prev和.carousel-control-next类创建， 并设置href属性为 Carousel 容器的id，data-slide属性指定了切换到上一个或下一个幻灯片的操作。 最后，在标签的末尾引入 Bootstrap 的 JavaScript 文件，包括 jQuery、Popper.js 和 Bootstrap.min.js。 通过使用 Bootstrap 框架，你可以很容易地创建一个具有响应式布局和动态切换效果的轮播图。 你可以根据需要自定义样式和调整轮播图的配置选项。
