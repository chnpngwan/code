## flea-market系统说明

#### 一、系统介绍

跳蚤市场（二手交易平台），分为后台管理员端和前台用户端两个部分。后台管理员可以对数据进行基本的管理统计操作，前台用户作为二手交易平台的客户可以拥有商家和客户两重身份，自行选择上架商品或购买商品。

#### 二、技术栈

后端：SpringBoot+MyBatis-Plus+JWT登录认证

前端：Vue2+ElementUI（router、axios、MD5等插件）

数据库：MySQL8.0
