## WordCount启蒙

**_老刘解读_**

`WordCount`作为大数据学习入门的经典启蒙案例，相当于学编程语言的`Hello World!`。其需求是实现统计一个英语文本的单词词频个数，比如：

源数据：

```
I always love CHINA!
I Always Love CHINA!
```

输出结果，区分大小写：
```
(I,2)
(always,1)
(Always,1)
(love,1)
(Love,1)
(CHINA!,2)
```
